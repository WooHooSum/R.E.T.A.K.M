This project was created for presentation during Opensauce thanks to the help of Hackclub.
I wanted to make a cyberdeck for some time and had even started work on two different ones using some spare parts I had laying around my room. Hackclub's grant let me take things a step farther and design what I wanted and not just what I could make with stuff I had laying around/could afford. Using that ability I designed this Cyberdeck based off of a design I saw on YouTube.

# R.E.T.A.K.M
RETAKM (re-take-em) stands for Remote Environment Tactical Assault Kit Manager. It's purpose is to allow for easy control of ATAK or similar software while also allowing rapid response prototyping. I wanted this to be out there so that other people could have a solid option for a cheap but highly capable workstation for in field work.
